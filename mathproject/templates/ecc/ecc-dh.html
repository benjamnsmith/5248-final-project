Messaging Encryption - Signal Protocol

Let's learn about how signal works. It uses EC Diffie Hellman! Well, a version of it. It uses triple DH.

For Alice to communicate her message to Bob, everyone will need a variety of public keys. Each of these has a corresponding private key, but as we know, if we can encrypt, we can decrypt, so we'll focus on encryption.

  1. $IK_A$ - Alice's identity key
  2. $EK_A$ - Alice's ephemeral key
  3. $IK_B$ - Bob's identity key
  4. $SPK_B$ - Bobs's (signed) prekey
  5. $OPK_A$ - Bob's one-time prekey
